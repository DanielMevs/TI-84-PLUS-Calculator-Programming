
This repository has code written on mmy TI-84 Plus calcultor. To write a program, press the prgmm button and look up blogs about TI BASIC
The code in this repository is mostly base conversion codes. 
First you'll need to download the TI Connect CE Software: https://education.ti.com/en/products/computer-software/ti-connect-ce-sw

Once you install the software and get the files from my github, you'll have to open up the newly downloaded software and click the button on the top with the icon of a calculator with an arrow pointing at itself and then select the file from my github
